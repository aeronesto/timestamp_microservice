Timestamp Microservice API

API accepts a date string as a parameter, and returns both the Unix timestamp and the natural language form of that date. Date must be either a unix timestamp or in natural language form, otherwise, it returns null.

API call:

https://aeronestos-timestamp-ms.glitch.me/September%1990,%200490
https://aeronestos-timestamp-ms.glitch.me/652406400

API output:

{"unix":652406400,"natural":"September 4, 1990"}
